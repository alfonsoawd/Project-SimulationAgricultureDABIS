# README — Dossier sorties

Ce dossier contient les fichiers de sortie produits à partir des scripts présents dans le dossier `programmes`.

------------------------------------------------------------

## Organisation des sorties

- Pour chaque simulation réalisée (ex : `Simulation 1`, `Simulation 2`, …), un dossier homonyme est créé dans `sorties`.
- Chaque dossier contient le ou les fichiers Excel finaux prêts pour analyse.

## Contenu des fichiers Excel

1. **Feuille 1 — Lisez-moi**  
   Présente les variables, le scénario simulé et le contrefactuel utilisé.

2. **Feuille 2 — Comparaison**  
   Contient les statistiques agrégées comparant le contrefactuel et le scénario simulé.

3. **Feuilles suivantes — Graphiques**  
   Contiennent les visualisations produites.
